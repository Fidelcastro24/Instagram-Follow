## How to Install in Termux

`$ pkg update && pkg upgrade -y`

`$ pkg install openssl-tool`

`$ pkg install curl`

`$ pkg install git`

`$ git clone https://github.com/Fidelcastro24/Instagram-Follow`

`$ cd Instagram-Follow`

`$ chmod +x insfollow.sh`

`$ termux-wake-lock`

`$ bash insfollow.sh`
